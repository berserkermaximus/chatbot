# AI chatbot
Simple chatbot using Ollama and FastAPI webhooks.

This uses a static `index.html` file to create new chats. Create multiple chat windows by opening multiple tabs of `index.html`.

# To start:
- Install `python 3.14` and `poetry`.
- Navigate to `backend` folder and run the command `poetry install`. This will install required packages.
- Run the command `./start.sh` to initialize the backend.
